### 🧠 Engineering Philosophy & Mental Models

I don't just build applications; I engineer systems. My approach is rooted in three core pillars that drive every line of code I commit:

*   **Performance First:** In both AI and Web, latency is a bug. Whether it's optimizing Agentic decision loops or model inference, speed is non-negotiable.
*   **Scalability by Design:** Architecting for 10 users is easy; architecting for 10,000 mandates modular microservices, clean Dockerization, and robust CI/CD.
*   **Data-Driven Logic:** Leveraging deep Data Structures and Algorithms to solve complex real-world problems with mathematical precision.
